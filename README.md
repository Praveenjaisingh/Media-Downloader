# 🎬 Media Downloader

A modern web application that allows users to download videos and audio from supported platforms by simply pasting a media URL. Built with a clean interface and optimized for speed, the application provides an easy way to access downloadable media formats.

🌐 **Live Demo:** https://media-downloader-mocha.vercel.app/

---

## ✨ Features

- 📥 Download videos from supported platforms
- 🎵 Extract and download audio (where available)
- 🔗 Paste URL to fetch downloadable formats
- ⚡ Fast and responsive interface
- 📱 Mobile-friendly design
- 🎨 Clean and modern UI
- 🚀 Lightweight and easy to use

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/media-downloader.git
```

### Navigate to the Project

```bash
cd media-downloader
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

## ▶️ Run the Application

Start the backend server:

```bash
npm run dev
```

Start the frontend:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 📸 How It Works

1. Copy the URL of a supported media.
2. Paste the URL into the application.
3. Click the download button.
4. Select the available format (if applicable).
5. Download the media to your device.
